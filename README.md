# Uber Trip Demand Analysis (April)

## Tools Used
- SQL (SQLite, DBeaver)
- Power BI (Web)
- GitHub

## Project Overview
Analyzed Uber trip demand for April to understand usage patterns,
base-wise contribution, and daily trends.

## Key Insights
- Over 564K Uber trips were recorded in April, indicating strong demand.
- Base B02682 contributes the highest share of trips.
- Trip volume fluctuates daily, with higher demand toward month-end.
